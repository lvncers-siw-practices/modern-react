# modern-react

To install dependencies:

```bash
bun install
```

To run the dev server (HMR enabled):

```bash
bun run dev
```

To run in production mode:

```bash
bun run start
```

サーバーが立ち上がったら、ターミナルに表示される URL (デフォルトは http://localhost:3000) を開くと `index.html` が見られるよ。

This project was created using `bun init` in bun v1.3.5. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
